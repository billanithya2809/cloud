🚀 Explore AWS Services for Data Analysis

AWS Services for Data Analysis

💻 Compute Services

Processing & Computation

  • Amazon EC2: Scalable virtual servers for data processing workloads
  • Amazon EMR: Managed Hadoop and Spark clusters for big data processing
  • AWS Lambda: Serverless computing for event-driven data processing
  • AWS Batch: Fully managed batch computing service
🗄️ Storage Services

Data Storage & Management

  • Amazon S3: Object storage for data lakes and backups
  • Amazon EFS: Managed file system for EC2 instances
  • Amazon EBS: Block storage for EC2 instances
  • AWS Storage Gateway: Hybrid cloud storage service
🗃️ Database Services

Managed Database Solutions

  • Amazon Redshift: Cloud data warehouse for analytics
  • Amazon RDS: Managed relational database service
  • Amazon DynamoDB: NoSQL database for real-time applications
  • Amazon DocumentDB: MongoDB-compatible document database
📊 Analytics Services

Data Analytics & Business Intelligence

  • Amazon Athena: Serverless query service for S3 data
  • Amazon QuickSight: Business intelligence and visualization
  • AWS Glue: Managed ETL service for data preparation
  • Amazon Kinesis: Real-time data streaming and analytics
🤖 ML & AI Services

Machine Learning & Artificial Intelligence

  • Amazon SageMaker: Fully managed ML platform
  • Amazon Comprehend: Natural language processing service
  • Amazon Rekognition: Image and video analysis
  • AWS DeepLens: Deep learning-enabled video camera
🔄 Data Pipeline Services

Data Movement & Integration

  • AWS Data Pipeline: Data workflow orchestration service
  • AWS DataSync: Data transfer service
  • Amazon Kinesis Data Firehose: Data delivery service
  • AWS Database Migration Service: Database migration tool
